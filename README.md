# Parcelador web

Web-based tool for interactive parcel subdivision and spatial distance analysis.

## Authors
- Félix González Peñaloza

## Institution
Institute of Natural Resources and Agrobiology of Seville (IRNAS-CSIC)

## Description
Parcelador web is a lightweight, client-side web application for generating and analysing land parcel layouts directly in the browser.  
It allows users to define parcel dimensions, separation distances, setbacks, and orientation, and to visualise and export the resulting geometries.

## Online version
https://TU_USUARIO.github.io/TU_REPOSITORIO/

## How to use
1. Open the web application.
2. Draw a polygon representing the parcel boundary.
3. Define parcel size, distances and orientation.
4. Generate parcels and export the results.
